# Mini Tweet App (Django Practice Project)

This is a **Mini Tweet Application** built using the Django framework.
The purpose of this project was **to practice Django fundamentals and understand how a complete Django application works**.

This project was created as part of my learning journey to explore **models, views, templates, forms, and CRUD operations in Django**.

---

## 📌 About the Project

The Mini Tweet App allows users to create and manage short tweets.
It is a simple project designed to practice building a web application using Django.

Through this project, I learned how different parts of a Django project work together to handle user interactions and database operations.

---

## 🚀 Features

* Create a tweet
* Edit an existing tweet
* Delete a tweet
* View all tweets
* Image upload support for tweets
* Django admin panel for managing data

## ⚙️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/your-username/your-repository-name.git
```

### 2. Go to the project directory

```
cd your-repository-name
```

### 3. Create virtual environment

```
python -m venv venv
```

### 4. Activate the environment

Windows:

```
venv\Scripts\activate
```

### 5. Install dependencies

```
pip install -r requirements.txt
```

### 6. Run migrations

```
python manage.py migrate
```

### 7. Start the server

```
python manage.py runserver
```

Open the browser and visit:

```
http://127.0.0.1:8000
```
